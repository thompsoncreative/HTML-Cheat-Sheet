HTML and CSS Cheat Sheet

Common HTML Tags

Tag Name Description
<html> HTML Represents the root (top-level element) of an HTML document, so it is also referred to as the root element. All other elements must be descendants of this element.
<head> Document Metadata Contains machine-readable information (metadata) about the document, like its title, scripts, and style sheets.
<title> Document Title Defines the document's title that is shown in a browser's title bar or a page's tab. It only contains text; tags within the element are ignored.
<body> Document Body Represents the content of an HTML document. There can be only one <body> element in a document.
<footer> Footer Represents a footer for its nearest ancestor sectioning content or sectioning root element. A <footer> typically contains information about the author of the section, copyright data or links to related documents.
<h1> Section Heading Represent six levels of section headings. <h1> is the highest section level and <h6> is the lowest.
<p> Paragraph Represents a paragraph. Paragraphs are usually represented in visual media as blocks of text separated from adjacent blocks by blank lines and/or first-line indentation, but HTML paragraphs can be any structural grouping of related content, such as images or form fields.
<div> Content Division Generic container for flow content. It has no effect on the content or layout until styled in some way using CSS (e.g. styling is directly applied to it, or some kind of layout model like Flexbox is applied to its parent element).

Other HTML Tags

Tag Name Description
<figure> Figure with optional caption Represents self-contained content, potentially with an optional caption, which is specified using the <figcaption> element. The figure, its caption, and its contents are referenced as a single unit.
<fig-caption> Figure Caption Represents a caption or legend describing the rest of the contents of its parent <figure> element.
<ol> Ordered List Represents an ordered list of items, typically rendered as a numbered list.
<ul> Unordered List Represents an unordered list of items, typically rendered as a bulleted list.
<li> List Item used to represent an item in a list. It must be contained in a parent element: (<ol>), an unordered list (<ul>), or a menu (<menu>).

Common CSS Selectors

Name Syntax Description
Universal * { style properties } The CSS universal selector (*) matches elements of any type.
Type element { style properties } The CSS type selector matches elements by node name. In other words, it selects all elements of the given type within a document.
Class .class_name { style properties } The CSS class selector matches elements based on the contents of their class attribute.
Id #id_value { style properties } The CSS ID selector matches an element based on the value of the element's id attribute. In order for the element to be selected, its id attribute must match exactly the value given in the selector.
Attribute [attr] Represents elements with an attribute name of attr. The CSS attribute selector matches elements based on the presence or value of a given attribute.
